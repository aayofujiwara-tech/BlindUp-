## 対応言語（i18n）

- 日本語（ja） / 英語（en）
- `messages/ja.json` + `messages/en.json` で管理
- URL: `/ja/...` `/en/...` に分離
- ヘッダー常設トグルボタン（JA | EN）で切替
- 言語設定は localStorage に保存
- TTS アナウンス言語も UI 言語に自動連動

## レイアウト仕様

### ブレークポイント

| 環境 | 幅 | レイアウト |
|------|----|-----------|
| モバイル縦 | 〜767px | 縦積み・大きめタイマー・下スワイプで設定 |
| モバイル横 | 〜1023px（landscape） | タイマー特化・シンプル表示 |
| タブレット | 768〜1023px | 横 2 カラム |
| PC / TV | 1024px〜 | 3 カラム構成・clamp() で 4K 対応 |

### 表示情報マトリクス

| 情報 | モバイル縦 | モバイル横 | PC / TV |
|------|-----------|-----------|---------|
| 残り時間 | 大 | **特大** | 大 |
| 現在ブラインド | ○ | **大** | ○ |
| 現在レベル | ○ | 小さく | ○ |
| 次レベルブラインド | ○ | **小さく（下端）** | ○ |
| 平均スタック | ○ | ✗ 非表示 | ○ |
| 賞金プール・人数 | ○ | ✗ 非表示 | ○ |
| 設定ボタン | ○ | 最小化 | ○ |

### モバイル横向き詳細
- 時間・ブラインドを中央にドーンと大きく（潔くシンプル）
- 次レベルブラインドは画面下端に `次: SB 300 / BB 600` と小さく 1 行
- 平均スタック等のサブ情報は非表示

### PC / TV（3 カラム）
- 左: レベル情報・次レベルプレビュー
- 中: タイマー（大）・現在ブラインド・プログレスバー
- 右: 平均スタック・プレイヤー数

### 縦横切替
- `screen.orientation` API で自動検知（`visualViewport` 併用で iOS 対応）
- 手動上書きボタンあり（localStorage に保存）
- CSS: `@media (orientation: landscape) and (max-width: 1023px)` でモバイル横向きを分離

## ブラインド設定

### 編集 UI
- テーブル形式で全レベルを一覧表示
- セル直接編集（スプレッドシートライク）
- レベル追加・削除・ドラッグ&ドロップ並び替え
- 休憩レベルをブラインド表と同じ画面で管理（行タイプ: `blind` / `break` を切替）
- アンティはレベルごとに個別設定

### プリセット設計原則
- 毎レベル約 33〜50% 増（急激なジャンプなし）
- アンティは BB の約 25% を目安に中盤から導入
- チップ単位: 25・50・100・500・1000・5000 で割り切れる数値のみ
- 終局目安: BB ÷ 総チップ量 ≥ 5% のレベル

## カスタムプリセット保存

| 項目 | 仕様 |
|------|------|
| 保存先 | Firebase Firestore |
| 認証 | Firebase Auth（Google / メール） |
| 上限 | 5個 / ユーザー |
| 付与情報 | 名前 + メモ（任意） |
| アクセス | ログイン済みなら複数デバイスで同期 |
| 未ログイン時 | プリセット選択・使用は可能、保存のみ不可 |

### Firestore データ構造

```
users / {uid} / presets / {presetId}
  - name: string          // プリセット名
  - memo: string          // メモ（任意）
  - levels: Level[]       // ブラインド構造
  - createdAt: timestamp
  - updatedAt: timestamp

// Level 型
{
  type: 'blind' | 'break'
  sb?: number
  bb?: number
  ante?: number
  duration: number  // 分
}
```

## 音声アナウンス

| 項目 | 仕様 |
|------|------|
| エンジン | Web Speech API（speechSynthesis）|
| タイミング | ① レベルアップ時　② 残り 1 分警告 |
| 言語 | UI 言語設定に連動（ja-JP / en-US） |
| コントロール | ON/OFF + 音量スライダー |
| iOS 対策 | ゲーム開始ボタン押下時にダミー発話でアクティベート |

### アナウンス文例

| タイミング | 日本語 | 英語 |
|-----------|--------|------|
| レベルアップ | 「レベル3 開始。スモールブラインド 200、ビッグブラインド 400」 | "Level 3 starting. Blinds 200 / 400" |
| 残り 1 分 | 「残り 1 分です」 | "One minute remaining" |
| 休憩開始 | 「休憩です。10 分間休憩します」 | "Break time. 10 minutes" |

## URL シェア

| 項目 | 仕様 |
|------|------|
| 含める内容 | ユーザーが選択（ブラインド構造 / 外観テーマ / 両方 をチェックボックス） |
| 圧縮方式 | lz-string で圧縮 → Base64 → `?cfg=` パラメータ |
| シェア方法 | URL コピー（ワンタップ）+ QR コード生成（qrcode.js） |
| URL スキーマ | `/ja/timer?cfg=BASE64` |
